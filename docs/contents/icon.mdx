## API Reference

### Icon

A wrapper component for Lucide React Native icons with theme support.

| Prop            | Type                               | Default   | Description                                   |
| --------------- | ---------------------------------- | --------- | --------------------------------------------- |
| `IconComponent` | `React.ComponentType<LucideProps>` | -         | The Lucide icon component to render.          |
| `size`          | `number`                           | `24`      | The size of the icon in pixels.               |
| `color`         | `string`                           | -         | Override color (takes precedence over theme). |
| `lightColor`    | `string`                           | -         | Color to use in light mode.                   |
| `darkColor`     | `string`                           | -         | Color to use in dark mode.                    |
| `strokeWidth`   | `number`                           | `1.8`     | The thickness of the icon strokes.            |
| `strokeLinecap` | `'round' \| 'butt' \| 'square'`    | `'round'` | The style of line endings.                    |

### Additional Props

The Icon component accepts all props from `LucideProps`, allowing full customization of Lucide icons.

## Theme Integration

The Icon component integrates with your app's theme system:

- Automatically uses theme colors when no explicit color is provided
- Supports separate colors for light and dark modes
- Falls back to the 'icon' theme color by default
- Color prop overrides theme colors when specified

## Best Practices

- Use consistent sizes throughout your app (16, 20, 24, 32px are common)
- Maintain proper contrast ratios for accessibility
- Use semantic icons that clearly represent their function
- Consider stroke width based on your design system
- Test icons in both light and dark themes

## Accessibility

The Icon component supports accessibility:

- Icons inherit accessibility properties from Lucide components
- Proper contrast is maintained with theme colors
- Icons scale appropriately with system text size settings
- Consider adding accessibility labels for interactive icons
